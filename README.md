# NoSpoon-Server

## Explanation
WS backend Server created to serve the `NoSpoon` game for `@fforres` talk :)

## Running
- `clone`
- `yarn install`
- `donwload && run redis`
- `yarn run watch`
  - `DEBUG=*,-nodemon* yarn run watch` for debugging in console
  - `yarn run watch:debug` for running dev-tool interactive debugger
